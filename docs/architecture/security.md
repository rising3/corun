# セキュリティ

機密情報管理:

- API キーやシークレットをコードやリポジトリにハードコードしない。
- 機密情報は環境変数、あるいはシークレットマネージャ（例: GitHub Secrets）で管理する。

権限管理:

- 実行ユーザー権限は最小権限で運用する（ファイルアクセスや外部サービス呼び出し時）。

実行時安全策:

- 外部コマンド実行時はインジェクション対策を行う（引数のバリデーション、適切なエスケープ）。
- サードパーティライブラリの依存は定期的に脆弱性スキャンを実施する。

監査と追跡:

- 重要操作はログに記録し、必要に応じて監査証跡を残す.
